# IsTool
Python tool that uses acidanthera's macserial to set-up iServices on Hackintoshes

***

## What this generates:
- MAC Address (Not used, only for ROM)
- ROM Number
- SystemSerialNumber
- MLB Number
- UUID (Using uuidgen)



***

## To install:

Do the following one line at a time in Terminal:

    git clone https://https://github.com/Zynth9999/IsTool
    cd IsTool
    chmod +x IsTool.command
    
Then run ```./IsTool.command``` or open with finder.
